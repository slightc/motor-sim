# 物理核心

## 相域模型

三相电压为输入，三相(αβ)电流为积分状态。电压方程在 αβ 下积分，避免 dq 的零序奇异。

## 凸极与交叉饱和

αβ 电感矩阵随转子角变化：
```
Sig=(Ld+Lq)/2, Dlt=(Ld-Lq)/2, Ldq=k_cross·iq
L = [[Sig+A, B],[B, Sig-A]],  A=Dlt·cos2θ - Ldq·sin2θ,  B=Dlt·sin2θ + Ldq·cos2θ
```
交叉饱和 Ldq 使凸极轴在高负载下偏移 φ_sat=½·atan(2·Ldq/(Ld−Lq))，这是 HFI 无感在高负载下的主要误差源。

## d 轴极性饱和（磁极辨识依据）

永磁体在 d 轴预偏置铁芯，使增量 Ld 随 i_d **不对称**：
```
Ld(i_d) = Ld0 · [1/(1+((i_d+i_pm)/i_knee)²)] / [1/(1+(i_pm/i_knee)²)]
```
- i_d=0 时 = Ld0
- +d(朝 N)磁通叠加→饱和→Ld 降低
- −d(朝 S)磁通相消→去饱和→Ld 升高

±脉冲电流响应的不对称即用于解 IPD 的 180° 模糊。`i_pm_sat=0` 时关闭，向后兼容。

## 铁损

Bertotti 三项(磁滞/涡流/异常)，依赖定子磁链 ψ_s；以制动转矩 P_fe/ω 计入机械方程。

## 关键工程经验

- 带宽分离：电流环 ≫ PLL ≫ 速度环。
- 死区用**真实电流**(物理)，补偿用**测量电流**(控制)。
- 解调用整周期相干累积；方波注入用相邻周期差(抵消基波斜率)。
