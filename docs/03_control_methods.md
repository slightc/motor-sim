# 控制方法

每个方法对应 `control/` 下一个可运行 demo。所有方法（除磁极辨识外）都是纯 Controller，不动 core。

## 速度/无感方法谱

| # | demo | 方法 | 适用区 | 关键点 |
|---|------|------|--------|--------|
| 01 | `01_foc_sensored` | 有感 FOC(编码器) | 全速域 | 基线，最可靠 |
| 02 | `02_sensorless_backemf` | 反电动势 PLL | 中高速 | 低速/参数失配失锁 |
| 03 | `03_sensorless_hfi_pulsating` | 脉振 HFI + 交叉饱和补偿 | 低速 | 带宽受慢低通限(~250Hz) |
| 04 | `04_sensorless_hfi_squarewave` | 方波 HFI | 低速 | 带宽~2.5kHz，零解调延迟 |
| 05 | `05_sensorless_ekf` | 扩展卡尔曼(含在线估 R) | 中高速 | 最优滤波抗噪 + 治参数漂移 |
| 06 | `06_startup_polarity_id` | IPD + 磁极辨识 | 零速启动 | 解 180° N/S 模糊(需 core 极性饱和) |
| 07 | `07_lowspeed_0to5` | 0–5 rad/s 对比 | 极低速 | EKF 失效，HFI 工作 |
| 08 | `08_fullrange_fusion` | HFI⊕EKF 速度自适应融合 | 全速域 | 按 ωeψ/(Ri) 可观测性加权 + 负载转矩观测器 |
| 09 | `09_if_opencontrol` | I/f 开环 + 主动阻尼 | 任意(兜底) | 无观测器→永不发散，效率低 |
| 10 | `10_position_servo_closedloop` | 闭环无感位置伺服 | 定位 | 级联+方波HFI，<1° 机械 |
| 11 | `11_multirate_timing` | 多速率时序 + 计算延迟 | 时序保真 | 物理细dt过采样PWM/控制按f_ctrl/ZOH/速度环分频/z^-N延迟 |

## 关键物理结论

- **无单一方法覆盖全速域**：零速只有凸极(注入)可观测，高速只有基波(反电动势)可行 → 必须融合(08)。
- **融合加权应按可观测性 ωeψ/(Ri) 而非纯速度**：高负载下交接点自动上移，对动态负载稳。
- **可靠性来自永不发散的兜底**：I/f(09) 无观测器，是分层容错的稳定基底。
- **角度定位精度 ≤ 估计精度**，但 1° 机械 = p° 电(p=4)，估计要求放宽 4 倍；闭环 FOC 比开环磁弹簧高效得多(按需取流)。

## 全速域可靠控制栈（推荐）

```
冷启动:   IPD(凸极) + 磁极辨识(极性饱和)              [06]
极低速:   方波 HFI + 交叉饱和补偿                      [04]
中高速:   EKF(在线估 R, 最优滤波抗噪)                  [05]
过渡:     按 ωeψ/(Ri) 加权融合 + 负载转矩观测器        [08]
兜底:     置信度低 → 退回 I/f(主动阻尼)                 [09]
```
